## 下载

### Docker

```
docker pull apachecn0/ibooker-ark
docker run -tid -p <port>:80 apachecn0/ibooker-ark
# 访问 http://localhost:{port} 查看文档
```



### NPM

```
npm install -g ibooker-ark
ibooker-ark <port>
# 访问 http://localhost:{port} 查看文档
```
